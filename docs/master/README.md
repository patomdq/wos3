## Cómo usar esto

Cuando analices una oportunidad, Claude leerá estos archivos para aplicar los criterios del máster:
- Identificación del tipo de operación (CCP, compraventa directa, etc.)
- Fiscalidad aplicable según el tipo de activo y operación
- Estructura de reparto de beneficios y liquidación
- Matriz de riesgos y checklist de análisis
